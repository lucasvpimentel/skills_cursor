# Code Architecture Planning

## Regra
Gerar somente passo a passo numerado.
Não gerar código.
Não executar nada.

## Cobertura mínima
- Definir objetivo técnico
- Definir responsabilidades por módulo
- Definir separação de camadas
- Definir contratos/interfaces
- Definir padrões arquiteturais (ex: clean architecture)
- Definir dependências externas
- Definir estratégia de testes
- Definir critérios de aceite técnico

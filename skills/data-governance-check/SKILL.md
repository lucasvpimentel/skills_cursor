# Governance Check

## Objetivo
Garantir uso responsável e auditável de dados.

## Procedimento
1. Classificar colunas:
   - PII/identificadores
   - sensíveis (saúde, financeiro etc.)
2. Minimização:
   - coletar/armazenar apenas o necessário
3. Proteção:
   - mascaramento/anonimização quando aplicável
   - segredos fora do código
4. Operação:
   - retenção e descarte
   - lineage (fonte -> transformação -> consumo)
   - controles de acesso
